DSP_YT-Playlist-Hoarder

About

A powerful tool for downloading and managing YouTube and Spotify content with an intuitive ASCII art interface.

Features

  • YouTube video and playlist downloads
  • Spotify track and playlist support
  • Quality selection options
  • Audio extraction capability
  • Playlist combination feature
  • Automatic media organization

Installation

Requirements:

  • Python 3.7+
  • FFmpeg
  • Internet connection

Setup:

python requirements.py
python youtube-playlist-hoarder.py

Usage

  1. Launch the program
  2. Paste a YouTube or Spotify URL
  3. Select your preferred options
  4. Wait for download to complete
